# 🚀 AI Resume Analyzer

An intelligent, full-stack web application designed to help job seekers perfectly tailor their resumes to their dream roles. By leveraging advanced AI, this tool evaluates your resume against specific job descriptions and provides a highly actionable, structured roadmap to bridge your skill gaps.

## ✨ Key Features

- **🧠 Smart AI Analysis**: Upload your resume (PDF) and a job description. The AI extracts the core requirements and evaluates your alignment.
- **🎯 Gap Analysis & Scoring**: Instantly receive a match percentage score and a clear breakdown of missing skills you need to acquire.
- **🗺️ 30-60-90 Day Roadmap**: Get a step-by-step, actionable learning plan generated by AI to help you master the missing skills over the next 3 months.
- **📂 Scan History**: Keep track of all your past resume scans, alignment scores, and expert recommendations in one organized dashboard.
- **🔒 Secure Authentication**: Fast and secure user authentication powered by Clerk.

## 🛠️ Technology Stack

**Frontend**
- **Next.js** (React framework)
- **Clerk** (Authentication)
- Modern, responsive custom CSS UI

**Backend**
- **Node.js** & **Express**
- **PostgreSQL** (Relational Database)
- **Prisma** (Next-generation ORM)
- **Google Gemini AI** (Core intelligent analysis engine via `@ai-sdk`)
- **Groq AI** (Secondary ultra-fast AI engine via `@ai-sdk`)

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites
- Node.js (v18 or higher)
- PostgreSQL database (or you can use SQLite locally)
- A Google Gemini API Key
- A Groq API Key (Optional for secondary AI engine)
- A Clerk account for Auth keys

### 1. Clone the Repository
```bash
git clone https://github.com/Krishna10016668/Resume-analyzer.git
cd Resume-analyzer
```

### 2. Backend Setup
Navigate to the root directory and install dependencies:
```bash
npm install
```

Create a `.env` file in the root directory with the following variables:
```env
PORT=3000
DATABASE_URL="postgresql://your_db_user:your_password@localhost:5432/resume_db"
GEMINI_API_KEY="your_gemini_api_key_here"
GROQ_API_KEY="your_groq_api_key_here"
```

Initialize the database:
```bash
npx prisma generate
npx prisma db push
```

Start the backend server:
```bash
npm run start
```

### 3. Frontend Setup
Open a new terminal window and navigate to the client folder:
```bash
cd client
npm install
```

Create a `.env.local` file inside the `client` folder:
```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY="your_clerk_publishable_key"
CLERK_SECRET_KEY="your_clerk_secret_key"
NEXT_PUBLIC_API_URL="http://localhost:3000"
```

Start the frontend development server:
```bash
npm run dev
```

The app will now be running on your local machine (usually at `http://localhost:3001` or `http://localhost:3000` depending on port availability).
